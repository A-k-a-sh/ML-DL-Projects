# CNN Fashion-MNIST Classification

Computer vision project using Convolutional Neural Networks (CNN) to classify clothing items from the Fashion-MNIST dataset.

## Overview
- Implements CNN architecture for image classification
- Uses the Fashion-MNIST dataset (clothing items)
- Converts PIL images and numpy arrays to PyTorch tensors
- Trains and evaluates CNN models for classifying 10 different clothing categories

## Dataset
- Fashion-MNIST: 28x28 grayscale images
- 10 classes of clothing items (t-shirts, trousers, dresses, etc.)
- Images automatically downloaded using torchvision

## Technologies Used
- PyTorch
- torchvision
- NumPy, Pandas
- Matplotlib, Seaborn
