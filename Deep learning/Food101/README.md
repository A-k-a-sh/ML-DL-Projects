# Food101 Image Classification

Deep learning project for classifying food images across 101 different food categories using the Food-101 dataset.

## Overview
- Multi-class image classification for 101 food categories
- Uses the Food-101 dataset with thousands of food images
- Implements deep learning models for food recognition
- Can identify various cuisines and dishes

## Dataset
- Food-101 Dataset
- 101 food categories
- Includes train and test splits
- Contains metadata with class labels and file paths
- Dataset structure includes JSON files for training/testing splits

## Technologies Used
- PyTorch
- NumPy, Pandas
- Matplotlib
- JSON for metadata processing

## Dataset Structure
- Training data: Organized by food categories
- Test data: Separate test set for evaluation
- Labels: 101 different food types
- Metadata includes train.json, test.json, and labels.txt
