# Binary and Multiclass Classification with PyTorch

This project demonstrates classification techniques using PyTorch on different datasets.

## Projects

### Binary Classification
- Creates a binary classification model using PyTorch
- Uses scikit-learn's `make_circles` dataset
- Implements neural networks to classify circular patterns
- Dataset: 1000 samples with noise
- Visualizes decision boundaries and model predictions

### Multiclass Classification
- Extends binary classification to multiple classes
- Implements multiclass neural networks
- Trains and evaluates models on multiclass datasets

## Technologies Used
- PyTorch
- scikit-learn
- NumPy, Pandas
- Matplotlib, Seaborn
