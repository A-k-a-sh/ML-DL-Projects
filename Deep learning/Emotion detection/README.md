# Facial Emotion Detection (Real-time)

Deep learning project for detecting facial emotions in real-time using Convolutional Neural Networks.

## Overview
- Detects and classifies facial emotions from images
- Implements CNN architecture for emotion recognition
- Trained on the Facial Emotion Recognition dataset
- Can be used for real-time emotion detection applications

## Dataset
- Facial Emotion Recognition Dataset
- Contains images labeled with different emotions (happy, sad, angry, surprise, etc.)
- Downloaded using Kaggle Hub

## Technologies Used
- PyTorch
- torchvision
- NumPy, Pandas
- Matplotlib, Seaborn
- ImageFolder for dataset loading

## Features
- Multi-class emotion classification
- Real-time emotion detection capability
- Data split into training and validation sets
- GPU acceleration support (CUDA)
